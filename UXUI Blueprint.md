## UX Structure
- Overview
- Ledger
- Reconciliation
- Reporting
- Governance
- Automation and Integrations

## UI Direction
- Warm-neutral atmospheric background with grid texture
- Mint signal accents for active and success states
- Rounded modular surfaces for dense operational data
- Sticky quick-navigation pills for module jumps
- Subtle motion for panel entry and hover feedback

## Frontend Mapping
- Navigation and hero shell: `apps/web/app/page.tsx`
- Module sections and panel hierarchy: `apps/web/app/page.tsx`
- Visual tokens, motion, and global component styles: `apps/web/app/globals.css`
